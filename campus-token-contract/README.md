# 🎓 Campus Quest

A gamified campus engagement platform built on Stellar Soroban.

## Features

- 🔐 Connect with Freighter Wallet
- 📝 Complete quests
- 🪙 Earn on-chain reward tokens
- 🎁 Redeem rewards
- 📱 Responsive design
- ⚡ Stellar Soroban smart contracts

## Tech Stack

- React + Vite
- Soroban (Rust)
- Stellar SDK
- Freighter Wallet
- Vercel
